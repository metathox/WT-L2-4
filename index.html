Що таке Git?

Git — це система контролю версій (VCS — Version Control System), створена у 2005 році Лінусом Торвальдсом. Вона дозволяє відстежувати зміни в коді чи файлах проєкту, бачити історію та повертатися до попередніх версій, а також працювати над проєктом кільком розробникам одночасно. Git сам по собі не зберігає файли у хмарі — для цього використовують GitHub, GitLab або інші сервіси.

Для чого потрібен Git?

  • Відстеження змін: кожна зміна файлу зберігається як commit, можна подивитися, що змінилося і коли.
  • Повернення до попередніх версій: якщо помилився або зламав щось, можна відкотитися назад.
  • Робота в команді: дозволяє декільком людям працювати над одним проєктом одночасно без конфліктів.
  • Експерименти без ризику: створюєш гілки (branches) для нових фіч, тестуєш їх, а потім об’єднуєш з основною версією.
  • Резервне копіювання: якщо проєкт зберігається на GitHub/GitLab, додані зміни не пропадуть.

  3. Mac OS:
    • sudo brew install git
